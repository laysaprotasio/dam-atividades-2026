# Save Pass 🔐

O **Save Pass** é um aplicativo gerenciador de senhas local desenvolvido em Flutter. 

## 📝 Atividade Prática DAM

Este repositório serve como base para a **Atividade de Navegação e Entrada de Dados**. 

Consulte as especificações detalhadas na atividade para implementar as seguintes funcionalidades:
*   Criação da tela de **Perfil do Usuário** (`ProfilePage`).
*   Configuração e validação de formulários com `TextFormField` e `FormState`.
